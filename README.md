# NudeFndr - Transparency Repository
Privacy-first iOS app for detecting sensitive content using Apple's on-device ML.
🔗 [nudefndr.com](https://nudefndr.com) | 📱 [App Store](https://apps.apple.com/jp/app/nudefndr/id6745149292)
🌍 English, Japanese, Thai, Chinese | 📄 MIT
## Included
Vault Encryption (ChaCha20), Keychain Integration, SensitiveContentAnalysis Wrapper
## NOT Included
UI, Business Logic, Production Code
## Security
On-device ML only. ChaCha20-Poly1305. Keychain storage. No network.
## Requirements
iOS 17+, Xcode 15+
