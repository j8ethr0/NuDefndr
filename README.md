## What's Included

- Vault Encryption (ChaCha20-Poly1305)
- Keychain Integration
- SensitiveContentAnalysis Wrapper

## What's NOT Included

- UI code
- Business logic
- Production app code

## Security

- On-device ML only (zero network)
- ChaCha20-Poly1305 encryption
- iOS Keychain storage (device-bound keys)

## Requirements

- iOS 17+
- Xcode 15+
